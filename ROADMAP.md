# Roadmap

This document outlines planned features and enhancements for future versions of Airbeeps.

## Planned Features

### Agent System
- **Agent Engine**: Tool calling and task execution capabilities
- **MCP (Model Context Protocol)**: Integration with external tools and services via MCP servers
- **Agent Tools**: Web search, API calls, and custom tool support
- **Agent Memory**: Experimental conversation memory for agents

### Authentication & User Management
- **OAuth Providers**: Google, GitHub, Microsoft, and other OAuth integrations
- **Email Verification**: User email verification system
- **Password Reset**: Email-based password recovery
- **SMTP Integration**: Full email notification support

### Storage & Deployment
- **S3-Compatible Storage**: Amazon S3, MinIO, and other S3-compatible backends
- **Docker Support**: Official Docker images and docker-compose configurations
- **Kubernetes Deployment**: Helm charts for production Kubernetes deployments
- **Cloud Deployment Guides**: AWS, GCP, Azure deployment documentation

### Platform Enhancements
- **API Documentation**: Interactive API documentation with examples
- **Webhooks**: Event notifications for integrations
- **Audit Logs**: Comprehensive activity logging for enterprise compliance
- **Multi-Tenancy**: Improved isolation for enterprise deployments
- **Analytics Dashboard**: Usage statistics and insights

### Performance & Scalability
- **Caching Layer**: Redis integration for improved performance
- **Async Task Queue**: Background job processing with Celery
- **Horizontal Scaling**: Load balancing and distributed deployment support
